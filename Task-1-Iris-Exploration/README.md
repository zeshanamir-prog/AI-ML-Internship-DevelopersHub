# Task 1: Exploring and Visualizing the Iris Dataset

## Objective
To explore and visualize the Iris dataset in order to understand feature
distributions and relationships using Python.

## Dataset
Iris Dataset (loaded using seaborn library)

## Tools and Libraries
- Python
- Pandas
- Seaborn
- Matplotlib

## Steps Performed
- Loaded and inspected the dataset
- Analyzed dataset structure and statistics
- Created scatter plots, histograms, and box plots
- Documented observations based on visual analysis

## Key Observations
- The dataset contains 150 rows and 5 columns
- No missing values were found
- Petal length and petal width clearly distinguish flower species
